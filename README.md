# Trackr - Job Tracker App

Trackr is a production-style job tracking application built with React and Tailwind CSS. It helps job seekers manage applications, track progress, organize jobs using a Kanban board, view analytics, and personalize their experience.

## Live Demo

[View Live Project](https://job-tracker-app-blue-mu.vercel.app/)

## Features

- Landing page with product-style sections
- Dashboard with clickable and filterable stats cards
- Applications page with drag-and-drop Kanban board
- Job status tracking: Applied, Interview, Offer, Rejected
- Search, filter, sort, hide, edit, and delete jobs
- Analytics page with bar chart and pie chart using Recharts
- Conversion rate and success metrics
- Insights with icons, confidence score, and animations
- Settings page with profile name and email
- Data persistence using localStorage
- Restore demo data and clear all applications
- Confirmation modal using Framer Motion
- Toast notifications for user feedback
- Dark and light mode using Tailwind class strategy
- Responsive landing page and app layout
